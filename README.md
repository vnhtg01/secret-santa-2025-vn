## Introduction

**Secret Santa 2025** web application - a fun and secret way to exchange gifts among your group of friends! With a beautiful interface, romantic snow effects and smart random logic, ensuring each person will have their own secret gift recipient.

## Features

- **Smart Random**: Each time you refresh the page, the system automatically creates a new random allocation
- **Secret Code Security**: Each player has their own secret code for authentication
- **Perfect Matching**: Ensures each person only gives to one person, each person only receives once
- **Snow Effect**: Beautiful interface with romantic snow falling effect
- **Random Messages**: Each time you open a gift, there will be different messages, creating surprises
